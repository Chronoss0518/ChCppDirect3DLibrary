# HLSLで利用できるHeaderFile群
## 概要

`HLSL`では`#include`を利用することができるため、外部の人でも作成したHLSLのライブラリを利用できるように用意している

## 使用方法

手順はShader側とC++側に分けて準備する必要があります。

### Shader側
- このフォルダ以降のファイルを`#include`する
- `#include`する場所より前に`#define`にて以下のいずれかを定義する
  - `_SM6_0_`
  - `_SM5_0_`
  - `_SM3_0_`
- `#include`する場所より前に`#define`にて`__SHADER__`を定義する

### C++側
- このフォルダ以降のファイルを`#include`する
- DirectXを用いてリソースのバインドを行う
